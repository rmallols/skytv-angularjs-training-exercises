Lesson 1

  1. Create a controller named 'UserCtrl'
  2. Inject the '$scope' to the controller (if possible, using array notation)
  3. Define a factory called 'users' with a method called 'get' that return an array of objects, each of them with the following fake information
    • Id
    • First name
    • Last name
    • Date of birth (dd-mm-yyyy)
  4. Inject the 'users' service in 'UserCtrl' and assign the array of users to the '$scope'
  5. Display the users in the screen (hint: use the 'ng-repeat' directive)
HINT: You can create both the controller and the factory on the lesson1Ctrl.js file. The components organization topic will be managed in the future.
